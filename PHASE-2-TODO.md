# PixelGraph 第二阶段收口清单

## 已完成

- ER / Class / Sequence / State / Flow 五类图可切换、编辑、预览、缩放、适应、全屏和导出。
- SQL ER 支持数据库视图、Chen ER 视图、节点拖拽、节点缩放、自动排版、字段显示控制、颜色和字号设置。
- ER 节点尺寸、位置、关系显示名、边标签文字与偏移已作为本地视觉状态保存，不回写 SQL。
- Chen ER 菱形关系文字可编辑，关系基数标签可编辑和拖动。
- Sequence 图已去掉 Mermaid 原生黑圈编号，改为业务层级编号；同步/异步通过实线/虚线表达。
- 默认模板、图表定义、主要中英文文案和导出错误提示已清理。
- 设置面板已有重置设置入口。
- Mermaid 与 ELK 自动排版已按需加载；`npm.cmd run build` 已验证通过。
- 数据库 ER 表节点字段过多时，字段区内部滚动，不再截断或撑破节点。

## 仍需人工验收

- 五类图分别验证 SVG、PNG、Markdown 导出。
- 浅色、深色、透明背景和 PNG 倍率分别验证一次。
- 桌面宽屏、移动窄屏、全屏模式分别检查控件遮挡和文字溢出。
- ER 验证字段很多的表、手动缩小节点、刷新恢复、自动排版、导出视觉状态。
- Sequence 验证层级编号、空心小人、同步/异步线型、导出一致性。

## 可后置增强

- 为 SQL 解析、ER model 转换、Sequence 编号、导出保护补最小测试。
- 模板切换前增加覆盖确认或撤销入口。
- Mermaid 错误提示继续细化到图类型声明、箭头语法、缩进或括号问题。
- 设置项按当前图类型继续收敛，只显示真正生效的配置。

## 不纳入第二阶段

- AI 生成、云同步、多人协作、桌面端、VS Code 插件。
- Monaco / CodeMirror 深度编辑器集成。
- Mermaid 单个节点拖拽缩放。
- PDF、JPEG、draw.io XML 等更多导出格式。
